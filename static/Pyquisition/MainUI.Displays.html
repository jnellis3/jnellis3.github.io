MainUI.Displays.Display_Widget module

class MainUI.Displays.Display_Widget.DisplayWidget(name: str, configDict: Optional[dict] = None, inputList=[])[source]

Bases: PyQt6.QtWidgets.QMdiSubWindow

Abstract class to be used as a base class for all widgets that will be displayed in the mdi area, excluding channel control widgets.

Parameters

  • name – Name of the widget, this can be any string, should be unique if it’s output will be used as an input for another widget.

  • configDict – Dictionary containing the configuration of the widget. If present, the restoreState method will be called to restore the widget to the state it was in when the dictionary was created.

  • inputList – List of input objects that can be used as inputs to the widget. This is used to populate the input comboboxes in the control widget. For example, if a widget is defined that takes a ‘Channel’ object as an input, then the inputList should contain a list of ‘Channel’ objects.

Variables

  • name – Name of the widget

  • inputs – Dictionary containing the inputs to the widget. Keys are the names of the inputs. The values are the input objects. The input objects can be any object, but should stay consistent across a subclass of DisplayWidget. For example, if a subclass of ‘PlotWidget’ is defined and listens to a ‘Channel’ object as an input, then the input object should be a ‘Channel’ object.

  • outputs – Dictionary containing the outputs of the widget. Keys are the names of the outputs. These values get updated after the process method is called. The output objects can then be used as inputs to other widgets.

MainUI.Displays.PlotWindow module

class MainUI.Displays.PlotWindow.PlotDisplayWidget(name: str, configDict: Optional[dict] = None, chList=None)[source]

Bases: MainUI.Displays.Display_Widget.DisplayWidget

This widget is capable of displaying outputs of channels contained in the activeChannels model. Can display multiple channels.

Parameters

  • name – Name of the widget

  • configDict – Dictionary containing the configuration of the widget. If present, the restoreState method is called

  • chList – List of channels that can be displayed in the widget. Will likely be the actChW

Variables

  • myType – String containing the type of the widget. This is used to identify the widget when it is saved to a file.

  • channels – List of channels that can be displayed in the widget. Will likely be the actChW

  • colors – List of colors that can be used to plot the data. The colors will be cycled through as more channels are added.

  • count – Number of data points that have been plotted. Used to determine when to clear the plot.

  • range – Number of data points that will be displayed on the plot at any given time.

  • pointer – Index of the most recent data point that has been plotted.

  • timer – Timer that calls the process method at a regular interval.
