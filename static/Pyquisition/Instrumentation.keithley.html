Available parameters to set. Should take the form of a dictionary of dictionaries, where the top level keys are the names of the settable parameters. Each parameter name should point to a dictionary containing information on that specific parameter. The instrument control box will use this dictionary to figure out what values may be changed. Each parameter dictionary MUST include, at minimum, “getter”, “setter”, and “dtype”. May also contain “limits”, a tuple indicating what the min and max allowed values are, and “allowable_value” which should contain a list of the possible inputs.

“getter” and “setter” should contain the actual functions used to get and set the parameter, while dtype is a string indicating the data type. May change “dtype” to an actual python type if I can get that to work properly.

“limits” should be a tuple with 2 values (either integers or floats), min value in position 0 and max in position 1 “dtype” should be the data type the setter expects. Should be a callable class or function.
